Homework 2: Question 3

Working example of running code is in Homework2.mov.

To run the code, follow these steps:
1. glfw-3.4 must be installed on your system.
2. modify the Makefile to point to the correct location of your glfw installation.
3. Open a terminal and navigate to the project directory.
4. Compile the code using the following command:
    make clean
    make 
    ./main

5. Use the mouse and keyboard to interact with the 3D object:
    - Up Arrow: Rotate up
    - Down Arrow: Rotate down
    - Left Arrow: Rotate left
    - Right Arrow: Rotate right
    - W: Zoom in
    - S: Zoom out
    - esc: Exit the program
